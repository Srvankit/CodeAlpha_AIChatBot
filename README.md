# CodeAlpha AI ChatBot

## Project Description

CodeAlpha AI ChatBot is a Java-based virtual assistant developed as part of the CodeAlpha Java Programming Internship.

The chatbot uses rule-based Natural Language Processing (NLP) techniques to provide interactive communication and utility features.

## Features

* Interactive conversation
* FAQ responses
* Current date and time
* Calculator
* Random jokes
* Learn new responses
* Chat history logging
* Persistent memory
* Session statistics

## Technologies Used

* Java
* Object-Oriented Programming (OOP)
* Java Collections Framework
* File Handling
* Exception Handling
* Date and Time API

## Project Structure

src/

* Main.java
* ChatBot.java
* KnowledgeBase.java
* Calculator.java
* ChatLogger.java
* SessionStats.java
* UI.java

data/

* responses.txt
* chat_history.txt

## How To Run

Compile:

javac src/*.java

Run:

java -cp src Main
